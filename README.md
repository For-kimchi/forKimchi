# 프로젝트명
> For 김치

---

## 개요
For 김치는 프로젝트는 김치 제조 공장의 생산 및 재고 관리를 위한 MES 시스템입니다.  
주요 기능:
- 영업 : 주문 등록 및 제품 입고, 납품 처리
- 생산 : 주문 기반 제품 생산
- 자재 : 자재 발주, 입고, 출고 처리
- 품질 : 자재 및 제품 검사

---

## 👥 팀원
| 이름 | 역할 | 담당 |
|------|------|--------|
| 김민식 | 팀장 | 기준관리 / 영업 / 배포 |
| 정유환 | 부팀장 | 생산 / 개발환경 |
| 김어진 | 팀원 | 자재 / Git |
| 이혁진 | 팀원 | 품질 / DB |


---

## ⚙️ 개발환경
- **Frontend**: Vue 3, Bootstrap 5, Pinia
- **Backend**: Node.js (Express), Axios,
- **Database**: MariaDB
- **Tools**: VS Code, Git, MySQL Workbench
- **배포환경**: Ubuntu, Nginx, PM2, NCP,
